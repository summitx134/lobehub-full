# LobeHub Full / AetherHub

AI Agent orchestration platform — "Your Chief Agent Operator" that organizes agents into 24/7 operation.

A forked and rebranded version of LobeHub with custom AetherHub branding, fission growth system, and full-stack deployment.

## Architecture
- **Web**: Next.js + Vite (SPA mode)
- **Desktop**: Electron + electron-builder
- **Mobile**: Expo (React Native)
- **Server**: Node.js backend
- **CLI**: Command-line interface
- **Database**: Drizzle ORM (PostgreSQL)

## Growth System (AetherHub Custom)
- Referral engine with cash-back, membership tiers, mentorship commissions
- Daily check-in system with streaks and milestones
- Task center (beginner, daily, growth tasks)
- Points/recharge center with bonus credits
- Lucky wheel with 10 prize types
- Leaderboards (invitation & spending rankings)
- Recall & group-buy campaigns
