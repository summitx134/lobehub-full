# Memory Commands
Manage user memories - the AI's long-term knowledge about users.

## Memory Categories
- identity: User's name, role, relationships
- activity: Recent activities and status
- context: Ongoing projects and goals
- experience: Past experiences and learnings
- preference: User preferences and directives

## lh memory list
List memory entries grouped by category.
## lh memory create
Create a new identity memory.
## lh memory persona
Display compiled memory persona summary.
