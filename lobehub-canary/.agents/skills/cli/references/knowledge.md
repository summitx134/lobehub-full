# Knowledge Base, File & Document Commands
Manage knowledge bases for RAG, uploaded files, and documents.

## lh kb list
List knowledge bases.
## lh kb create
Create a knowledge base with name and description.
## lh kb upload
Upload a local file to a knowledge base via S3 presigned URL.
## lh doc create
Create a document with title and body.
## lh file list
List uploaded files.
